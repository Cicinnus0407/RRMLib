Retrofit2+RxJava+MVP和常用工具类封装库
##用法
- 方式1
    - newest-version = 1.0.3
    - Android Studio
        ```
        com.cicinnus.RRMLib:rrmlib:newest-version'
        ```
- 方式2
    - Android Studio导入lib
    -
        ```
        compile project(':rrmlib')
        ```
